# AOC2022
My solution to the Advent of Code 2022

## Installation

```bash
# clone the repository
git clone https://github.com/SvenSZim/AdventOfCode.git
cd AdventOfCode/2022

# choose the day
cd day01

# insert your inputdata
mkdir data
# paste your data in this folder "data.txt"

# run solution
python3 main.py
```
