# AOC2024
My solution to the Advent of Code 2024

## Installation

```bash
# clone the repository
git clone https://github.com/SvenSZim/AdventOfCode.git
cd AdventOfCode/2024

# choose the day
cd day01

# init cmake
cmake .

# build
cmake --build .

# insert your inputdata
mkdir data
# paste your data in this folder "data.txt"

# run solution
./solution
```
